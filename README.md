# 🎮 Minecraft-like Telegram Multiplayer Bot

> ⚠️ **Требование**: Для корректной работы эмодзи необходим **Telegram Premium** у создателя бота

## 📋 Описание проекта

Мультиплеерный Telegram-бот в стиле Minecraft с онлайн-миром, где несколько игроков могут взаимодействовать одновременно в реальном времени. отображение с помощью premium emoji.

## 🌟 Особенности

- **👥 Мультиплеер в реальном времени** - до 20 игроков в одном мире
- **❤️‍🩹 10 сердечек здоровья** - система жизней и атаки других игроков
- **🎲 Случайный спавн** - каждый игрок появляется в радиусе 50 блоков
- **🏗️ Интерактивный мир** - ломайте и ставьте блоки
- **🧱 Несколько типов блоков** - камень, трава, дерево
- **📊 Система высот** - меняйте уровень игрока (2D+)

## 🕹️ Управление

### Основные действия:
```

• ⬆️⬇️⬅️➡️ - Движение по миру
• ⛏️ - Ломать блоки / Атаковать игроков
• 🪨 - Поставить каменный блок
• 🌿 - Поставить блок травы
• 🪵 - Поставить деревянный блок
• 🔺🔻 - Изменить высоту персонажа

```

### 📱 Команды бота:
| Команда | Описание | Эмодзи |
|---------|----------|---------|
| `/start` | Главное меню | 🏠 |
| `/newgame` | Начать новую игру | 🆕 |
| `/continue` | Продолжить сохраненную игру | ▶️ |
| `/join` | Присоединиться к существующему миру | 👥 |
| `/help` | Справка по управлению | ❓ |

## 🎯 Игровая механика

### 💖 Система здоровья:
- **❤️ 10 сердечек** у каждого игрока
- **⚔️ Атака других игроков** кнопкой ⛏️
- **💀 При смерти** - телепорт на случайную точку спавна
- **💚 Авторегенерация** со временем

### 🌍 Мир:
- **📏 Размер мира:** 40×40 блоков
- **📶 Три уровня высоты**
- **🧱 4 типа блоков:** воздух, камень, трава, дерево и ресурсы, руды.
- **💾 Сохранение состояния** мира между сессиями

### 👥 Мультиплеер:
- **🌐 Все игроки** в одном экземпляре мира
- **⏱️ Реальное время** обновления позиций
- **👀 Видимость других игроков** на карте
- **🔄 Общие ресурсы** и территория
